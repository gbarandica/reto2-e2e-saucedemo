# Reto 2 - Estrategia de Automatización End-to-End SauceDemo

Este repositorio contiene la implementación del Reto 2 de automatización E2E sobre la aplicación SauceDemo.

## Herramientas utilizadas

- Serenity BDD con Java
- Playwright con Python
- Behave para BDD en Python
- Jenkins para CI/CD
- GitHub como repositorio
